# Ziphp

[![PHP 8.0+](https://img.shields.io/badge/PHP-8.0+-blue.svg)](https://www.php.net/downloads)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

**Ziphp** is a set of microframework-like utilities for developing PHP applications quickly and easily using view routers, resource routers, models, controllers, layouts, views, and partials.

## Installation

```bash
composer require davidleonstr/ziphp:dev-main --prefer-source
```
